# Hyakumonogatari

Japanese storytelling app with audio recording and AI transcription.

## Features
- Create and manage storytelling events
- Record audio for each story
- AI-powered transcription using OpenAI Whisper
- Visual candle-blowing ceremony

## Setup
1. Install this application
2. Open the Setup UI
3. Configure PostgreSQL connection
4. Configure OpenAI API key
5. Start the service
